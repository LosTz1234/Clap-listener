# Clap-listener
This code listens for claps and taps the screen when read. The delay is minimal, about 0.3 seconds. Instructions: turn on shizuku, grant access to Termux. Next, you need to enable rish. Place the clap.py file and the other files exported from shizuku in a folder accessible to Termux, and then run the file through it: python clap.py.
